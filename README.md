# CloudCompliance-Lab

This project simulates validating a GxP-compliant cloud infrastructure using AWS and Azure.

## Structure
- `terraform/`: Infrastructure as Code for AWS and Azure
- `validation/`: IQ/OQ/PQ templates and validation reports
- `compliance/`: Audit mappings and change request templates
- `scripts/`: Automation scripts for provisioning
- `.github/`: CI/CD workflows for validation tracking
